# Kushagra Agrawal — Portfolio

## About

Full-Stack & AI Developer portfolio showcasing projects, skills, experience, and contact information.

## Getting Started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
